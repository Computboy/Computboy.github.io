# GAMES101

## Overview

A structured learning record for the foundations of modern computer graphics.

## Key Concepts

- Linear algebra and geometric transformations.
- Rasterization and antialiasing.
- Shading, texture mapping, and geometry.
- Ray tracing, sampling, and physically based rendering.
