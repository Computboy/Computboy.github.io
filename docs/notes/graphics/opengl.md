# OpenGL

## Overview

Practical notes on building and debugging real-time graphics applications with OpenGL and GLSL.

## Key Concepts

- Context setup and the rendering loop.
- VAO, VBO, EBO, textures, and framebuffer objects.
- Shader compilation, linking, uniforms, and interface contracts.
- Model loading, render-state management, and diagnostic tooling.

## Notes

The notes will connect API-level behavior to the [Realtime 3D Scene Editor](../../projects/realtime-3d-scene-editor.md), with emphasis on ownership, state transitions, and debugging rather than isolated snippets.
