# Rendering Pipeline

## Overview

A systems-level view of how geometry becomes an image, from transforms and clipping through rasterization, shading, visibility, and post-processing.

## Key Concepts

- Object, world, view, clip, and screen spaces.
- Primitive assembly, clipping, rasterization, and interpolation.
- Depth testing, blending, and framebuffer operations.
- Lighting, shadow mapping, normal mapping, and screen-space effects.

## Notes

This page will combine mathematical derivations with observations from both GPU-based OpenGL work and the [Tiny Software Rasterizer](../../projects/tiny-software-rasterizer.md).
