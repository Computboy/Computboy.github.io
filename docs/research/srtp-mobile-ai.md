# Mobile AI SRTP

## Overview

A research record for lightweight super-resolution and frame interpolation in mobile inference scenarios.

## Research Questions

- Which architectural choices provide the best quality-to-latency tradeoff?
- How should spatial fidelity and temporal stability be evaluated together?
- What changes when models are measured on target devices rather than desktop hardware?

## Experiments

Dataset preparation, baseline models, loss functions, metrics, conversion steps, and device measurements will be recorded here as reproducible experiment entries.

## Current Status

The current phase focuses on defining baselines and a consistent evaluation pipeline before optimizing model structure.
