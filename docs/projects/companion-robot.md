# Companion Robot for Children

<p class="project-meta"><strong>Stack:</strong> Arduino Nano · Embedded Control · Ultrasonic Sensing · Physical Prototyping</p>

## Overview

An earlier industrial-design and embedded-systems project that reworked the Otto Basic platform into a friendlier companion-oriented robot for children.

## Motivation

The project explored how physical form, expressive motion, and responsive behavior could make a small educational robot feel more engaging than a conventional programmed toy.

## What I Built

I redesigned the appearance and structure, developed the Arduino Nano control system, and connected user instruction, sensing, onboard processing, and module output into one interaction chain.

## System Architecture

User input flows through voice and distance-sensing modules into command parsing and state switching. The onboard controller coordinates movement and feedback modules based on the active interaction context.

## Core Features

- Voice interaction.
- Ultrasonic distance sensing.
- Context-dependent motion and feedback behaviors.
- Multi-module state coordination.

## Technical Implementation

The embedded program covers sensor reading, command parsing, state transitions, motion behavior programming, and coordination between perception and output modules.

The central challenge was making hardware, motion, and interaction logic behave as one coherent system under limited embedded resources.

## Reflection

The project established an early foundation in end-to-end interactive systems: input, sensing, state, response, and physical expression.
