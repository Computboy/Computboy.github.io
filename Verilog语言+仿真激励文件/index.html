    仿真激励文件+verilog语法

    一、Verilog电路描述语言

    先给定一个Verilog文件（顶层设计）的模板：

    module main( I0,
             I1,
             I2,
             O );

   input I0;
   input I1;
   input I2;

   output O;

   wire s_logisimNet0;
   wire s_logisimNet1;
   wire s_logisimNet10;
   wire s_logisimNet2;
   wire s_logisimNet3;
   wire s_logisimNet4;
   wire s_logisimNet5;
   wire s_logisimNet6;
   wire s_logisimNet7;
   wire s_logisimNet8;
   wire s_logisimNet9;

   assign s_logisimNet0 = I0;
   assign s_logisimNet1 = I1;
   assign s_logisimNet4 = I2;


   assign O = s_logisimNet9;

   // NOT Gate
   assign s_logisimNet7 = ~s_logisimNet0;

   // NOT Gate
   assign s_logisimNet3 = ~s_logisimNet1;

   // NOT Gate
   assign s_logisimNet6 = ~s_logisimNet4;

   AND_GATE_3_INPUTS #(.BubblesMask(3'b000))
      GATES_1 (.input1(s_logisimNet0),
               .input2(s_logisimNet3),
               .input3(s_logisimNet6),
               .result(s_logisimNet10));

   AND_GATE_3_INPUTS #(.BubblesMask(3'b000))
      GATES_2 (.input1(s_logisimNet7),
               .input2(s_logisimNet1),
               .input3(s_logisimNet6),
               .result(s_logisimNet5));

   AND_GATE_3_INPUTS #(.BubblesMask(3'b000))
      GATES_3 (.input1(s_logisimNet7),
               .input2(s_logisimNet3),
               .input3(s_logisimNet4),
               .result(s_logisimNet2));

   AND_GATE_3_INPUTS #(.BubblesMask(3'b000))
      GATES_4 (.input1(s_logisimNet0),
               .input2(s_logisimNet1),
               .input3(s_logisimNet4),
               .result(s_logisimNet8));

   OR_GATE_4_INPUTS #(.BubblesMask(4'h0))    //其中.BubbleMask括号中的3＆4和子文件中一致
      GATES_5 (.input1(s_logisimNet10),
               .input2(s_logisimNet5),
               .input3(s_logisimNet2),
               .input4(s_logisimNet8),
               .result(s_logisimNet9));
endmodule

    1.输入输出模块

    module main( I0,
             I1,
             I2,
             O );

   input I0;
   input I1;
   input I2;

   output O;

    这段语句标定了模拟电路的输入和输入模块，从module开始，main表示文件名称，随后定义了三个输入元器件和一个输出元器件

    2.wire语句

       wire s_logisimNet0;
   wire s_logisimNet1;
   wire s_logisimNet10;
   wire s_logisimNet2;
   wire s_logisimNet3;
   wire s_logisimNet4;
   wire s_logisimNet5;
   wire s_logisimNet6;
   wire s_logisimNet7;
   wire s_logisimNet8;
   wire s_logisimNet9;

    wire语句将我们在Logisim中绘制的每一条线路进行定义并给定名称
    可将wire看作是一根导线，他存在输入和输出两端

    3.assign语句

    assign模块将wire进行连接。
    I0、I1、I2 作为 input 可以认为是特殊的 wire，它的输入端要保留给外部连接模块的电路，所以只有它们的输出端可以在模块内部使用

    0中间有斜横杠，O（大写字母o）中间没有斜横杠，一般表示的就是输出元器件！！

       assign s_logisimNet0 = I0;
   assign s_logisimNet1 = I1;
   assign s_logisimNet4 = I2;


   assign O = s_logisimNet9;

    assign s_logisimNet0 = I0;
    表示将s_logisimNet0和I0相连——I0的输出端连接0号导线，也同时说明了0号导线的输出端信号和I0输入元器件的输入端信号模式/大小相同

    在 Verilog 中，输入信号通常是由外部电路或模块提供的，它们的值通常是在等式右侧。而输出信号通常是由当前模块驱动的，它们的值应该在等式左侧。

       // NOT Gate
   assign s_logisimNet7 = ~s_logisimNet0;

    这是逻辑取反的标志，NOT门操作（通常为反相器）

    此外，我们也可以将一位立即数1或0直接赋值（输入）到模拟电路中：
    一位立即数的表示方式为 1'bX，其中：
    1 表示位宽是 1（即一位），
    'b 表示后面的数字是二进制形式，
    X 是该位的值，可以是 0 或 1。
    所以，1'b1 和 1'b0 分别表示：
    1'b1：一个一位的立即数，值为 1，通常对应高电平 Vcc。
    1'b0：一个一位的立即数，值为 0，通常对应低电平 GND。

    4.与门、或门、模块的调用

    • Verilog中的模块其实可以简单类比成C语言中的函数

    模块就是某一电路组成，对于 AND3 门电路和 OR4 门电路 Logisim 生成了 AND_GATE_3_INPUTS 模块（定义在 verilog/gates/AND_GATE_3_INPUTS.v）（相当于是多文件程序中的子文件，其中对该函数有阐释说明———对应着模拟电路中的一个小器件）和 OR_GATE_4_INPUTS 模块（定义在 verilog/gates/OR_GATE_4_INPUTS.v）

        module AND_GATE_3_INPUTS(
        input input1,
        input input2,
        input input3,
        output result
    );

    该段程序定义在外部文件verilog/gates/AND_GATE_3_INPUTS.v中，命名方式显示出这是一个有3个输入的与门。

    如何来调用这个模块？

       AND_GATE_3_INPUTS #(.BubblesMask(3'b000))
      GATES_1 (.input1(s_logisimNet0),
               .input2(s_logisimNet3),
               .input3(s_logisimNet6),
               .result(s_logisimNet10));

    GATES_1是模块名称

    .input1(s_logisimNet0) 这条语句的作用和 assign s_logisimNet0 = imput1 相似，都是将线路0和与门的第一个输入端口进行连接。

    .result(s_logisimNet10)表示将与门的输出端口和线路10相连接。

    每根导线只能有1个输入，可以有0到多个输出，.result()这样的悬空写法是无害且必要的。

    二、激励仿真模块

    • testbench.v：驱动电路并观察输出的激励（高电平刺激）仿真模块
    module Testbench;

    reg I0,I1,I2;   //寄存器，能够提供稳定输出的变量
    wire O;

    initial begin
        // 设置输入的值
        I0 = 1'b0; I1 = 1'b0; I2 = 1'b0;   //一位立即数
        // 等待 5 个单位时间修改输入激励的值
        #5;
        I0 = 1'b0; I1 = 1'b0; I2 = 1'b1;
        ...
        #5;
        // 结束仿真
        $finish;
    end

    main dut( 
        .I0(I0),
        .I1(I1),
        .I2(I2),
        .O(O) 
    );
    // 被仿真的模块，接受不同的输入，然后观察输出是否正确

    `ifdef VERILATE
        initial begin
            // 生成波形文件 build/verilator/Testbench.vcd
            $dumpfile({`TOP_DIR, "/Testbench.vcd"});
            //记录 dut 模块以及内部模块的线路信号变化
            $dumpvars(0, dut);
            // 开始记录信号
            $dumpon;
        end
    `endif

endmodule

    三、进阶版Verilog代码的实现

    1.用与、或、非运算符来替代与、或、非门

    2.线组

    可以用 a[i] 依次索引 a 中的各根线路，其中i∈[NUM1,NUM2]∪[NUM2,NUM1]，数据类型为 wire。

        wire [0:0] a;
    //从左到右依次为 a[0]
    wire [3:0] a;
    //从左到右依次为 a[3] a[2] a[1] a[0]
    wire [0:3] a;
    //从左到右依次为 a[0] a[1] a[2] a[3]
    wire [7:1] a;
    //从左到右依次是 a[7] a[6] a[5] a[4] a[3] a[2] a[1]
    wire [3:9] a;
    //从左到右依次是 a[3] a[4] a[5] a[6] a[7] a[8] a[9]

    可以用 a[ i : j ] 索引 wire 数组的一组连续的线路，i,j∈[NUM1,NUM2]∪[NUM2,NUM1]。下面的代码首先将 a[7:0]的 a[2]、a[3]、a[4] 取出，然后从左到右组合起来得到 a[2:4]。

        wire [7:0] a;
    //从左到右依次是 a[7] a[6] a[5] a[4] a[3] a[2] a[1] a[0]
    a[2:4];
    //从左到右依次是 a[2] a[3] a[4]
    a[4:2];
    //从左到右依次是 a[4] a[3] a[2]
